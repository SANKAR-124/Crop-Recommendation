# 🌾 **SmartCrop: AI-Powered Crop Recommendation & Profitability Analysis System**

## 📋 **PROJECT OVERVIEW**

### **Problem Statement**
Farmers often make crop selection decisions based on intuition or tradition, leading to:
- Suboptimal yields due to soil-crop mismatch
- Economic losses from choosing low-profit crops
- Inefficient use of soil nutrients
- Lack of data-driven decision support

### **Solution**
SmartCrop is an intelligent decision support system that recommends the top 3 most suitable crops based on soil parameters and environmental conditions, with:
- Confidence scoring for each recommendation
- Economic profitability analysis
- Explainable AI showing why each crop was recommended
- Soil profile clustering for personalized insights
- Feature importance visualization

```python
def calculate_revenue(crop_name):
```

```python
    },

        'P': (40, 60),
        'K': (20, 40),
        'temperature': (15, 25),
        'humidity': (50, 70),

        'rainfall': (50, 100),
        'primary_drivers': ['temperature', 'rainfall']
    },
    # ... for all 22 crops
}
```

**Explanation Generation Logic:**
```python
def generate_explanation(crop, user_input):
    profile = CROP_PROFILES[crop]
    explanations = []
    
    for feature, optimal_range in profile.items():
        if feature == 'primary_drivers':
            continue
            
        user_value = user_input[feature]
        min_val, max_val = optimal_range
        
        if min_val <= user_value <= max_val:
            explanations.append(f"✅ {feature.capitalize()}: {user_value} (optimal range: {min_val}-{max_val})")
        elif abs(user_value - min_val) < (max_val - min_val) * 0.2:
            explanations.append(f"⚠️ {feature.capitalize()}: {user_value} (slightly below optimal: {min_val}-{max_val})")
        elif abs(user_value - max_val) < (max_val - min_val) * 0.2:
            explanations.append(f"⚠️ {feature.capitalize()}: {user_value} (slightly above optimal: {min_val}-{max_val})")
        else:
            explanations.append(f"❌ {feature.capitalize()}: {user_value} (outside optimal range: {min_val}-{max_val})")
    
    # Highlight primary drivers
    primary = profile['primary_drivers']
    
    return {
        'all_factors': explanations,
        'key_drivers': [f"🔑 {driver.capitalize()} is a primary factor for {crop}" for driver in primary]
    }
```
